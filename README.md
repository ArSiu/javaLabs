# javaLabs

### Task rewrite code from lab2(UML for python) on java using:
- maven,
- FindBugs,
- CheckStyle,
- PDM,
- Jacoco.
### hOw tO Run
```
$ git clone https://github.com/ArSiu/javaLabs.git
$ cd javaLabs
$ git checkout lab2
$ mvn compile
$ mvn site
$ cd target/site
$ run checkstyle.html
```
